Here's all the fun projects that I have made.

Health Risk Prediction

Developed a framework to enhance health prediction with the revised fusion node and deep learning paradigms. We used bayesian fusions to estimate the probability of occurrence of a disease based on the evidence in the past medical history of the patient and with the help of a neural network, derived the essential features

Chat Bot

Developed a Chat Bot using Keras and NLTK which asks a few questions to the user based on the theory of psychosexual development by Sigmund Freud and then records it to categorize personalities. It then interacts with the user based on their personality type as inputted.

Recommendation System

Lead a team of 3 people to develop an online tool to screen various reviews and categorize them by identifying the adjectives used in the review. We built a library in python in which we assigned scores to different adjectives based on the intensity. We extracted data from web pages using a web crawler and parsed the HTML data and used Naive Bayes algorithm to classify the reviews and based on the library, calculated the final score.
